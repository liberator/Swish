Pomoc -> Jes-Soft Basketball Playbook

Oficjalna strona firmy Jes-Soft
Swish jako zewnętrzną aplikację do tworzenia zagrywek wykorzystuje program Basketball Playbook (wersję 010 lub 009) autorstwa Jes-Soft. Aplikacja ta jest dostępna za darmo do ściągnięcia z oficjalnej strony lub stąd.

Obsługa aplikacji jest bardzo prosta - polecamy obejrzeć krótkie demo flash.

Eksport zagrywki do formatów jpg i flash

Aby wyeksportować zagrywkę do formatu jpg wybierz File -> Export -> Image 200%. Następnie wybierz gdzie zapisać obrazek oraz zaznacz format wyjściowy JPG.

Aby wyeksportować zagrywkę do formatu flash wybierz File -> Export -> Play -> zakładka Flash. Odznacz opcję "Within html page", kliknij OK i wybierz gdzie zapisać plik wynikowy.
